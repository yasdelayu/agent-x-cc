# Priority Scorer

## Назначение
Оценивает и ранжирует задачи для оптимального распределения времени и токенов.

## Критерии скоринга (каждый 1-10)

| Критерий | Вес | Описание |
|----------|-----|---------|
| Срочность | 3x | Дедлайн, запрос Босса = макс |
| Важность | 3x | Влияние на проект/доход |
| Стоимость токенов | 2x | Инверсия: дешевле = выше |
| Потенциал заработка | 2x | Для маркетплейса |
| Соответствие навыкам | 1x | Из SKILLS.md |

## Формула
```
score = (срочность × 3 + важность × 3 + (10 - стоимость) × 2 + заработок × 2 + навыки × 1) / 11
```

## Приоритеты (жёсткие правила)
1. Запрос Босса — всегда первый
2. Безопасность — второй
3. Оплаченная работа — третий
4. Автономные задачи — по скорингу

## Инструменты
- Plane MCP — статусы задач
- token_budget.py — остаток бюджета
- work_engine.py — статус маркетплейса
