# Report Formatter

## Назначение
Стандартизированные шаблоны для всех типов отчётов. Обеспечивает консистентный формат для владельца.

## Шаблоны

### Утренний брифинг (09:00)
```
Доброе утро.

Сервер: [статус]
Задачи: [кол-во active в Plane]
Бюджет: [остаток токенов]
[1-2 ключевых события]

Планы на день:
- [пункт]
- [пункт]
```

### Ежедневный отчёт (21:30)
```
Отчёт за ДД.ММ

1. [выполненная задача]
2. [выполненная задача]
...

Маркетплейс: [статус/заработок]
Токены: [использовано/лимит]
```

### Еженедельный обзор (Пн 10:00)
```
Неделя ДД.ММ — ДД.ММ

Выполнено: X задач
Маркетплейс: $Y заработано
Токены: Z использовано

Топ достижения:
1. [пункт]
2. [пункт]

Планы на неделю:
1. [пункт]
2. [пункт]

Требует решения владельца:
- [пункт] (если есть)
```

### Обзор соцсетей агентов (20:00)
```
Соцсети агентов ДД.ММ

Новости: [кол-во]
- [новость]

Возможности: [кол-во]
- [возможность]

Инциденты: [кол-во]
- [инцидент] (если есть)
```

## Правила форматирования
- Без эмодзи (если владелец не просит)
- Кратко: 3-7 пунктов
- Telegram HTML: `<b>`, `<code>`, `<pre>`
- Максимум 4096 символов (лимит Telegram)
- Числа и метрики — точные, не округлённые
