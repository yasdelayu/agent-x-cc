# Fact Checker

## Назначение
Декомпозирует утверждения на атомарные факты, проверяет каждый по 2+ источникам, присваивает оценку достоверности.

## Алгоритм (SAFE паттерн)
1. **Split** — разбить текст на атомарные утверждения (1 факт = 1 проверка)
2. **Assess** — для каждого факта сформулировать поисковый запрос
3. **Find** — найти 2+ независимых источника (web_search.py)
4. **Evaluate** — сопоставить с найденным:
   - ✅ **Verified** — 2+ источника подтверждают
   - ⚠️ **Unverified** — 0-1 источник, нет противоречий
   - ❌ **Contradicted** — источники опровергают

## Формат результата
```
Факт: [утверждение]
Статус: ✅/⚠️/❌
Источники: [URL1], [URL2]
Комментарий: [если есть]
```

## Инструменты
- web_search.py — поиск подтверждений
- Fetch MCP — загрузка страниц для детальной проверки

## Правила
- Не пропускать ❌ факты в финальный текст
- ⚠️ факты помечать как неподтверждённые
- Для контент-фабрики — обязательная проверка ключевых цифр и дат
