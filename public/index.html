Paid in USDC, over x402

x402 revives the dormant HTTP 402 Payment Required status into a real payment rail. The buyer signs a gasless EIP-3009 transferWithAuthorization; a facilitator verifies and settles it on-chain. AgentX holds no private key.

CHALLENGE

402 + requirements

An unpaid buy returns 402 with the exact USDC amount, asset and payTo.

SIGN

EIP-3009 authorization

Buyer's wallet signs a gasless USDC transfer — no gas, no approval tx.

SETTLE

Facilitator settles

Coinbase CDP facilitator submits it on Base; server verifies the receipt.

MINT

Credits minted

USDC lands in payTo; X402 credits mint to the account.

// an agent pays automatically with the x402-fetch client:
import { wrapFetchWithPayment } from "x402-fetch";
import { privateKeyToAccount } from "viem/accounts";

const account = privateKeyToAccount(process.env.PRIVATE_KEY);
const fetchWithPay = wrapFetchWithPayment(fetch, account);

// 402 challenge is handled + paid transparently → 200 + minted credits
const res = await fetchWithPay(
  "https://agentx.example/api/x402/buy/pro?account=agent:acme-corp"
);
console.log(await res.json()); // { ok:true, settlement:{ transaction, payer }, balance }
